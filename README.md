# time-series-forecasting-sales

The global sales of specific books is tracked from 1st January 2012 - 31st December 2024. 

Techniques include:
                   ## Classical diagnostic methods (ACF/PACF, ADF, Ljung-Box, differencing)
                   ## Modelling (Auto-ARIMA/SARIMA/SARIMAX)
                   ## Machine-learning (XGBoost) 
                   ## SARIMA+LSTM Hybrid (sequential and parallel)
